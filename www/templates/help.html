Getting Started with WiTTS

There are three prerequisites to using the WiTTS system:
  • An NFC enabled mobile device with the WiTTS Mobile Application installed
  • A valid user account in Back&
  • A WiTTS Temperature Sensor
For additional information on how to obtain all of the above, please refer to "Support".

Measure & Record Temperature

'Measure & Record Temperature' can be accessed through the 'Home' tab. This feature effectively measures the patient's temperature and presents an option for the user to record the temperature (in Fahrenheit) under the patient's ID number in the patient records database hosted on Back&. To begin, simply tap the WiTTS Temperature Sensor near the NFC sensor of the mobile device. Upon successful connection with the device, the following information will be displayed:
  • Patient name
  • Date of birth
  • Assigned nurse
  • Temperature last taken
  • Measured temperature (Fahrenheit)

*** Note ***
  • A second scan may be required on first use of the application due to an unresolved issue in the NFC library. Please see "Known Issues"
  • Information fetched from the Back& server may be delayed as a result of an unstable or slow internet connection. In this case, please wait a few seconds or clear the record and try again.

 Once the user confirms the information presented, s/he may proceed to record the temperature associated with the patient by pressing the 'Record' button. A dialog box will appear to inform the user of a successful transmission.
